# Archive

This folder stores historical or exploratory materials that are not canonical product requirements.

## Policy

1. Archived files are read-only historical context.
2. Active planning and implementation must use canonical docs from [docs/README.md](../README.md).
3. If a historical note still contains useful constraints, migrate those constraints into a canonical doc and keep the archive entry unchanged.

## Archived items

- `opuslast.md`: historical session transcript log.
- `research.md`: exploratory market/product research draft.
- `dead-buttons-session-10.md`: historical snapshot from Session 10 audit.
